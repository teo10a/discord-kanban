# 🚀 퍼:레이드 업무공간 팔로우업 칸반보드

디스코드(Discord) 포럼 채널의 스레드를 연동하여 팀의 업무 진행 상황을 시각적인 칸반 보드 형태로 제공하는 실시간 웹 대시보드입니다.

## ✨ 주요 기능
- **실시간 양방향 동기화**: 디스코드와 웹 보드가 실시간으로 동기화됩니다. (스레드 생성, 메시지 작성, 태그 변경 등)
- **드래그 앤 드롭 (Drag & Drop)**: 웹에서 카드를 이동시키면 디스코드 스레드의 태그(우선순위/분류/상태)가 즉시 변경됩니다.
- **다양한 보기 모드**: 전체 보기(3열 그리드), 우선순위별 보기, 분류별 보기 토글 지원
- **봇 명령어 연동**: 스레드 내에서 디스코드 명령어를 통해 업무 메타데이터 업데이트 (`!요약`, `!담당자` 등)
- **일자별 업무 일지**: 스레드 상세 모달에서 날짜별로 업무 기록 및 수정/삭제 가능 (데이터 영구 저장)
- **반응형 웹 디자인**: 데스크탑, 태블릿, 모바일 스마트폰 환경 완벽 지원

## 📋 사전 준비 (Prerequisites)
- Node.js (v16.9.0 이상 권장, Discord.js v14 기반)
- 디스코드 개발자 포털에서 생성한 **디스코드 봇 토큰 (Bot Token)**
- 칸반 보드와 연동할 **디스코드 포럼 채널 ID**

## ⚙️ 설치 및 설정 방법 (Installation & Setup)

### 1. 패키지 설치
프로젝트 폴더에서 터미널을 열고 아래 명령어를 실행하여 필요한 라이브러리를 설치합니다.
```bash
npm install
```

### 2. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env` 파일을 생성하고, 다음 내용을 입력합니다.
```env
DISCORD_TOKEN=당신의_디스코드_봇_토큰을_여기에_입력하세요
FORUM_CHANNEL_ID=연동할_포럼_채널의_ID를_여기에_입력하세요
```

### 3. 디스코드 봇 권한 및 인텐트(Intents) 설정
디스코드 개발자 포털(Developer Portal)에서 봇에 다음 권한들을 반드시 부여해야 합니다.
- **Privileged Gateway Intents**: `Message Content Intent` (메시지 내용 읽기) 활성화 필수
- **Bot Permissions** (서버 초대 시 필수 권한):
  - `View Channels` (채널 보기)
  - `Send Messages` (메시지 보내기)
  - `Send Messages in Threads` (스레드에서 메시지 보내기)
  - `Read Message History` (메시지 기록 보기)
  - `Manage Threads` (스레드 관리 - **웹에서 드래그 앤 드롭으로 태그를 변경하기 위해 필수!**)

## ▶️ 실행 방법 (Running the App)

1. 터미널에서 아래 명령어를 입력하여 백엔드 서버를 실행합니다.
```bash
node server.js
```
2. 웹 브라우저를 열고 다음 로컬 서버 주소로 접속합니다.
```text
http://localhost:3001
```

## 💬 봇 명령어 가이드 (디스코드 스레드 내부에서 사용)
디스코드 포럼의 각 스레드 채팅창에 아래 명령어를 입력하면 웹 칸반 보드 카드에 즉시 반영됩니다.
- `!요약 [내용]` : 해당 업무의 요약 설명 작성
- `!담당자 [정담당자] [부담당자]` : 업무 담당자 배지 지정 (예: `!담당자 김철수 박영희`)
- `!팀원 [이름1] [이름2] ...` : 참여 팀원 목록 지정
- `!업무일지 [내용]` : 오늘의 업무 일지 추가 (웹 모달에서도 직접 작성 가능)
- `!경고기준 [일수]` : 지정한 일수 동안 새 메시지가 없으면 웹 카드에 미활동 경고 표시 (기본값: 3)
